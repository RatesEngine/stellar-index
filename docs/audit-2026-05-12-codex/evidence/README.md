# Evidence Directory

Evidence must be primary, reproducible, and tied to source or runtime
observations.

Files:

- [log.md](log.md): evidence ledger
- [commands.md](commands.md): command transcript summaries
- [cross-file-interactions.md](cross-file-interactions.md): system
  coupling ledger
- [r1-runtime.md](r1-runtime.md): SSH/R1 runtime observations

Do not paste secrets, private keys, tokens, full env dumps, or raw
customer data. Redact sensitive output and say what was redacted.
