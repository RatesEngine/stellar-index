# Journey Evidence (supporting)

The journey **traces** themselves live one level up at
`../../journeys-traces/J##-<short-name>.md`.

This subdirectory is for *supporting* evidence that backs a
journey trace but is too large/structured for the trace file
itself: e.g. raw query outputs, decoded XDR fixtures, captured
HTTP traces.

Naming: `J##-<topic>-<YYYYMMDD>.md`.
