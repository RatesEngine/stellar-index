# Workstream Evidence (supporting)

Per-workstream working notes and supporting evidence that don't
fit the master `../log.md` ledger row.

Naming: `W##-<topic>-<YYYYMMDD>.md` (or just `W##-<topic>.md`
for a long-running notebook).

The workstream **sub-plans** themselves live at
`../../workstreams/W##-<title>.md`.
